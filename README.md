# Meu-Portif-lio
Portfólio pessoal desenvolvido com HTML, CSS e JavaScript, focado em acessibilidade (WCAG), responsividade e boas práticas de UI/UX. Inclui menu lateral acessível, navegação por teclado, foco visível, alto contraste opcional e semântica estruturada.

Seu site está online em https://lucasavelarbr.github.io/Meu-Portif-lio/
